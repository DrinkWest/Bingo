Bingo

1. Run the "_CreateDatabase.sql" script on a database server to create a database named Bingo located in D:\default\
2. [Optional] Run the "_AddWords.sql" script to add a list of default words to the database.
3. Setup an IIS site and copy all the files (except _AddWords.sql, _CreateDatabase.sql, _Readme.txt)
4. Edit default.asp and configure with the database settings
5. Go to the website
